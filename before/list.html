«До»: список Objects (63)

Текущий вид раздела Objects на demo.racktables.org. Числа на карточке — ключевые проблемы, мешающие быстро выполнять основные задачи (поиск, фильтрация, понимание статуса, переход к действию).

  1. 1. Поиск спрятан в шапке и не фильтрует таблицу инкрементально — пользователю проще нажать Ctrl+F в браузере. Поиск по asset tag/IP не очевиден.
  2. 2. Единственный фильтр — по тегам. Нет фильтров по типу (Server/Switch/PDU…), площадке (Row/Rack), статусу (Has problems). Пользователь скроллит 63+ строки руками.
  3. 3. Колонки заголовков не сортируются и не имеют визуальных индикаторов сортировки. Невозможно быстро упорядочить, например, по asset tag или по стойке.
  4. 4. «Has problems» = только подсветка строки — без иконки, легенды и без причины. На втором экране пользователь не помнит, что значит красный фон.
  5. 5. Хлебные крошки минимальны и не дают переход «назад в стойку» одним кликом. Нет breadcrumbs к связанным сущностям.
